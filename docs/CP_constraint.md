근무표 작성 기준

[주기/기본]
-   스케줄 단위: 28일 고정 주기(cycle)
-   주기 시작일(실제 달력 기준): <MM/DD(요일)> (예:3/1(일))
-   주기 종료일: 시작일+27일
-   주기 내 일수 28일(고정)
-   주기 내 날짜 표기 방식
   내부 계산용: day1~day28
   출력 시: 실제 달력 날짜로 작성 <MM/DD(요일) (예:3/1(일))

[하드 제약 목록]
-   근무시간은 1일 8시간, 주 40시간
   D(데이): 6am-2pm
   E(이브닝): 2pm-10pm
   N(나이트): 10pm-6am(+1)
-   H-01: : D/E/N 일별 정원 준수 (각각 D=<2>, E=<2>, N=<2>) 
-   H-02: Night는 반드시 2~3일 연속 블록으로만 배정 (1일 N 금지, 4일 이상 연속 N 금지)
-   H-03: N 블록 종료 후 연속 2일은 근무 금지(OFF/주휴만 가능) 
   허용 예: N→OFF→OFF / N→주휴→OFF / N→OFF→주휴 
-   H-04: 5일 초과 연속 근무 금지 (D/E/N을 근무로, OFF/주휴는 연속 끊음) 
-   H-05: 근무 간 최소 16시간 휴식 
   - 금지 전환: N→D, E→D 
-   H-06: 상기 지정 근무 불가 준수 
-   H-07: 주휴는 OFF와 구분하여 배정하며, 먼저 고정 후 나머지 근무 배정

[주휴(하드)]
-   주휴는 “주휴”로 표기
-   주기 규칙: 1주기 고정, 2주기부터 하루씩 당김 (예: 1주기 화요일, 2주기 월요일)
-   주휴는 주기 내에서 바꿀 수 없음
-   간호사별 고정 주휴 있음 
   <간호사1>: <매주 월요일>
   <간호사2>: <매주 화요일>
   <간호사3>: <매주 수요일>

[소프트 제약(점수 최소화)] 
-   S-01: 개인별 D/E/N 개수 균형(평균 대비 편차 <2) 
-   S-02: 개인별 비근무일 개수 형평성 
-   S-03: 주휴 ±1일에 OFF 인접 시 보너스

[가중치(원하는 비중으로 숫자 조정)] 
-   balance_shift(기본 5): <5> 
-   balance_off(기본 5): <5> 
-   rest_adjacent_bonus(기본 -2): <숫자>

[비근무일 관련 기준]
-   비근무일은 “주휴”, “OFF”, “생휴”, “VAC”, “수면”, “법휴”, “공가”로 규정
-   일주일(일요일부터 토요일) 안에 “주휴” 1개, “OFF” 1개를 필수로 부여해야 한다
-   “주휴”는 바꿀 수 없다
-   이외 비근무일 발생 시 “VAC”으로 입력한다
-   부여된 “VAC” 중, 한 달에 한 개의 “VAC”은 “생휴”로 바꿀 수 있다
-   법정 공휴일에 “OFF” 또는 “VAC”이 부여된 경우, “OFF” 또는 “VAC”을 “법휴”로 바꾼다
-   일주일(일요일부터 토요일) 중, “법휴”가 발생한 경우, “법휴”를 제외하고 “주휴”와 “OFF”가 하나씩 필수로 부여되어야 한다 
-   일주일(일요일부터 토요일) 중, “공가”가 발생한 경우, “공가”를 제외하고 “주휴”와 “OFF”가 하나씩 필수로 부여되어야 한다. 
-   부여된 “VAC” 중, 한 달에 한 개의 “VAC”은 “수면”으로 바꿀 수 있다. 단, 아래 기준을 충족할 때에만 “수면”으로 변경 가능하다. 
   (홀수)월에 7개 이상의 N 근무를 한 경우
   (홀수)월 또는 (홀수+1)월에 “수면” 1개 발생
   (홀수)월에 7개 이상의 N 근무를 한 뒤, (홀수+1) 월에 7개 이상의 N 근무를 한 경우
   (홀수)월 또는 (홀수+1)월에 “수면” 1개 발생
   (홀수+1)월 또는 (홀수+2)월에 “수면” 1개 추가 발생
   (홀수)월에 7개 이상의 N 근무를 한 뒤, (홀수+1) 월에 7개 미만의 N를 근무한 경우
   (홀수)월 또는 (홀수+1)월에 “수면” 1개 발생
   (홀수+1)월 또는 (홀수+2)월에 “수면” 미발생
   (홀수)월에 7개 미만의 N 근무를 한 경우
   (홀수)월 N 합산과 (홀수+1)월 N 합산이 11일 이상일 경우
   (홀수+1)월 또는 (홀수+2)월에 “수면” 1개 발생
